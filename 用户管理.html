Ubuntu 16.04


创建用户：
	create user 'gsmrlab'@'%' identified by 'password';

查看权限：
	show grants for gsmrlab;
	
授权：
	grant select,update,insert on FwsDb.*  to gsmrlab;
	grant all privileges on *.* to gsmrlab;

刷新：
	flush privileges 
	
	
远程登陆：
	mysql配置：
		/etc/mysql/mysql.conf.d/mysqld.cnf 注释掉：bind-address        = 127.0.0.1
	用户设置：
		update user set host='%' where user='gsmrlab';
	关闭防火墙：
		service iptables stop

Ubuntu 20.04

安装：
	apt install mysql-server mysql-client libmysqlclient-dev -y
	
更改mysql默认root用户密码:
	ALTER USER 'root'@'localhost' IDENTIFIED WITH mysql_native_password BY '新密码';
	FLUSH PRIVILEGES;

创建用户：
	CREATE USER '新增用户名'@'localhost' IDENTIFIED WITH mysql_native_password BY '密码';
	GRANT ALL PRIVILEGES ON *.* TO '新增用户名'@'localhost' WITH GRANT OPTION;
	FLUSH PRIVILEGES;
	
修改默认加密方式：
	default_authentication_plugin=mysql_native_password;
