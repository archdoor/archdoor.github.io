数学表达式

  • + - * / **
  • 整数和浮点数：7/2=3 (不是3.5)
  • 复数：{3,5} = 3+5i
  • 数学函数：几乎同Ｃ语言函数
  • 自定义函数：f(x)=x+1、f(x,y)=x+y
  • 圆周率(π)：pi

简单函数绘图

  1. plot sin(5*x)：发现图中有折线，不圆滑
  2. set samples 500：设置采样率（默认100,100，第二个值在3D中有效）
  3. replot：重新绘画
  4. unset key：去除函数说明
  5. set title "sin(5*x)"：设置标题(大多数""和''没区别，但特殊字符如'\n'，单引号表示一般字符，双引号就是则会转意)
  6. set xlabel "X"：设置x轴标签
  7. set ylabel "Y"：设置y轴标签
